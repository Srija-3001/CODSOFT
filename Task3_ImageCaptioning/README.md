# Image Captioning AI

## Objective
Generate captions for images using Artificial Intelligence.

## Technologies Used
- Python
- Transformers
- BLIP Model
- Pillow

## How It Works
1. Input image is provided.
2. BLIP model extracts image features.
3. Language model generates a caption.
4. Caption is displayed to the user.

## Output Example
Generated Caption:
a dog running through a grassy field
