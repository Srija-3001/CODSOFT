# Rule-Based Chatbot

## Description
This project is a simple Rule-Based Chatbot developed using Python.
The chatbot responds to user inputs using predefined rules and pattern matching techniques.

## Features
- Greetings
- Date and Time
- AI and Machine Learning Information
- Python Information
- Simple Calculator
- Help Command
- Exit Command

## Technologies Used
- Python
- Regular Expressions (re)

## How to Run
1. Install Python
2. Run chatbot.py
3. Start chatting with the bot
