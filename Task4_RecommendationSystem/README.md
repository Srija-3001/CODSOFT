# Movie Recommendation System

## Description

This project is a content-based movie recommendation system developed using Python. It recommends movies based on genre similarity using cosine similarity and CountVectorizer.

## Features

* Movie recommendations
* Content-based filtering
* Cosine similarity
* Easy command-line interface

## Technologies Used

* Python
* Pandas
* Scikit-Learn

## Files

* recommendation_system.py
* movies.csv
