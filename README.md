# CODSOFT - Artificial Intelligence Internship

This repository contains the projects completed during my Artificial Intelligence Internship at CodSoft.

## Projects

### Task 1 - Rule-Based Chatbot

A chatbot that responds to user queries using predefined rules and pattern matching techniques.

### Task 2 - Tic-Tac-Toe AI

An AI-powered Tic-Tac-Toe game developed using Python and Tkinter with Minimax Algorithm.

### Task 3 - Image Captioning

An AI system that generates captions for images using a pre-trained transformer-based model.

### Task 4 - Recommendation System

A content-based movie recommendation system that suggests similar movies using cosine similarity.

## Technologies Used

* Python

* Pandas

* Transformers
* PyTorch
* Computer Vision
