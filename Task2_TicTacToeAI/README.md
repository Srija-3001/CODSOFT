# Tic-Tac-Toe AI

## Description

This project is an AI-powered Tic-Tac-Toe game developed using Python and Tkinter. The project includes a console version, a GUI version, and an advanced AI version using the Minimax algorithm.

## Features

* Human vs Human gameplay (Console Version)
* Human vs AI gameplay (GUI Version)
* Minimax Algorithm for intelligent AI moves
* Restart Game functionality
* Win, Loss, and Draw detection

## Technologies Used

* Python
* Tkinter
* Minimax Algorithm

## Project Files

* `tic_tac_toe.py` - Basic console version
* `tic_tac_toe_gui.py` - GUI version
* `tic_tac_toe_minimax.py` - AI-powered Minimax version
